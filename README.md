# Unreal-Blender Sync

This workspace contains starter scaffolding for:

- `unreal-plugin/` — an Unreal Engine 5.8 plugin skeleton for exporting skeletal meshes, hair, materials, skin weights, and animation to Blender.
- `blender-addon/` — a Blender extension scaffold with one-click export/import operators, auto-weighting support, and a framework for AI/MCP integration.

## Goals

- Seamless export between Unreal Engine 5.8 and Blender
- One-click export of mesh, skinning, hair, materials, and animation
- Blender-side auto weight painting and easy retargeting hooks
- Modular architecture for future feature upgrades
- AI extensibility via Blender MCP integration

## Notes

This is a starting point. The generated code is a scaffold and requires Unreal Engine 5.8 and Blender environment setup for testing and refinement.
